Typographic Compostitions

depictions of a passage about language

The prompt given was to create a typographic composition using an approximately 80-word passage about language – that’s it. With little direction, I was challenged to create pieces that were visually interesting, making sure to be intentional with my grouping and spacing of words and letters as well as my use of negative space. Initially, I was allowed to use only one font, font size, and weight, and after some time I was able to incorporate an additional font size. This project was really interesting and freeing, even though there were a number of constraints.
